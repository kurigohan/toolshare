Fequently asked questions

What is a shed? A shed is a place in which a user can store tools or borrower tools. Each share zone has at least one shed. When a user create an account their home is a representative shed. A Shed is a physical location that a user could goto received their borrowed tools.
What happens when I borrow? By borrowing you request from another user a specific tool they own if they approve you, then you can go to the shed that holds that tool or the user home and receive it. If they do not approve you, then the tool remains with the owner. At the end of a specific time period you must return the tool to the shed or owner of the tool.
