FAQ

What is a shed? A shed is a place in which a user can store tools or borrow tools. Each share zone has at least one shed. When a user creates an account their home is a representative shed. A shed is a physical location that a user can go to in order to receive their borrowed tools.
What happens when I borrow? By borrowing you request from another user a specific tool they own. If they approve you, then you can go to the shed that holds that tool or the user home and receive it. If they do not approve you, then the tool remains with the owner. At the end of a specific time period you must return the tool to the shed or owner of the tool.
What is the difference between a shared tool and a loaned tool? A shared tool is a tool with no borrower; that is, a tool ready to share with any borrower. A loaned tool is currently on loan to some borrower. If you borrow a tool and return it, please hit the "Return" button on its description so others can see that it is available again.
